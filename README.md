# Spirit Island Companion
This App provides a way to randomize the setup for the boardgame Spirit Island. You can assign spirits and adversaries and also manage the invader deck.

# Features
- Generate a random map setup with spirits for up to 6 players
- Generate a random adversary for different difficulty levels including scenarios
- Manage the invader deck
